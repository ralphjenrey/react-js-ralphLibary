# REACT JS LIBRARY MANAGEMENT SYSTEM
### Tech Used
1. React JS
2. Node JS
3. Firebase
4. Material UI

## Features 
- Role Based Authentication (Admin, Staff, User)
- Account Management (CRUD)
- Book Management  (Display, CRUD)
- Borrow and return features

## Additional Features
- Dark Mode
